block

Aug 30, 2016 | Hits

（一）简介

  • block用来封装一段代码，可以在任何时候执行
  • block的标志：^
  • block跟函数很像：
    1> 都有返回值
    2> 都可以保存代码
    3> 有形参
    4> 调用方法一样
  • block和函数的区别：
    1> 函数只能在方法外面定义
    2> block可以定义在方法内，也可以定义在方法外

（二）block定义:

1.没有返回值、没有形参，可以省略后面形参的括号

void (^block变量名)() = ^{

}

2.无返回值、有参数的

3.有返回值、有形参传入

返回值类型 (^block变量名)(形参类型) = ^(形参名)
{
​
}

（三）block访问外部变量

1.block内部可以访问外部的变量
2.默认情况下，block内部不能修改外部变量
3.给局部变量加上__block关键字，这个局部变量就可以在block内部修改
